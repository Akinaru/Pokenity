# Character Images

Put all character images in this folder.

- Path: `front/app/src/main/assets/characters/`
- Use the exact file names you save in the back-office/API (`avatarFileName` and `imageFileName`).
- Example: if a character has `imageFileName = red_main.png`, the file must exist at:
  `front/app/src/main/assets/characters/red_main.png`

Supported extensions by API validation:
- `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`, `.svg`
