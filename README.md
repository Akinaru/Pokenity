# Pokenity

## Demarrage du projet

Suivre ces etapes dans cet ordre:

1. Cloner le projet.
2. Lancer les services Docker:

```bash
docker compose up -d
```

3. Acceder a la base de donnees via [http://localhost:81](http://localhost:81).
4. Se connecter avec:
   - user: `root`
   - pass: laisser le champ vide
5. Creer la base `pokenity`.
6. Importer la base de donnees qui se trouve a la racine du projet.
7. Ouvrir le projet Android dans le dossier `front/` avec Android Studio.
8. Lancer le projet depuis Android Studio.
